# Style guide

Back to spec: **[CLAUDE.md](../CLAUDE.md)**

## Visual targets
- Dark, slightly blurred background art
- Semi-transparent dark overlay for readability
- Purple/blue borders for segmented controls
- Selected segment: purple fill
- CTA: orange button

## Typography
- Section headers: uppercase, medium weight, tracking slightly wider
- Segment numbers: large, bold
- Life total: very large, bold, centered

### Player panels
- Minimal chrome
- Center life total
- `-` left and `+` right
- Rotate entire panel via `RotatedBox`

## Spacing
- Use `SafeArea`
- Use consistent padding around controls
- Avoid UI elements near screen edges
